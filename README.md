# Ashay Kushwaha — Portfolio

Personal portfolio for Ashay Kushwaha — a systems builder and open-source engineer.

**Stack:** Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · GSAP · Vanta.NET · Lucide

## Dev

```bash
npm install
npm run dev     # → localhost:3000
npm run build   # → static export (out/)
```
